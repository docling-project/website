venue: IEEE International Parallel and Distributed Processing Symposium (IPDPS …
summary: "A unified framework of stochastic matrix-function estimators allowing computation of a subset of elements of the matrix f(A), where f is an arbitrary function and A is the adjacency matrix of the graph. The framework offers significant computational advantages for large-scale data where the computational cost is proportional to the size of the subset, contrary to the traditional O(N³) from diagonalization. The approach is naturally scalable with multiple compute nodes and performs well on accelerators."
