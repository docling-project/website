title: Enhancing multi-threaded sparse matrix multiplication for knowledge graph oriented algorithms and analytics

date: 2024-06-01
venue: IBM Research  (3), 1-40
summary: "Develops optimized algorithms for multi-threaded sparse matrix-matrix multiplication tailored for knowledge graph-oriented workloads. Sparse matrix multiplication is a critical operation with applications in scientific computing and graph analysis. The paper develops parallel algorithms with focus on performance portability across different high-performance computing architectures. Develops a meta-algorithm to choose the right algorithm and data structure based on problem characteristics. Performance comparisons across architectures demonstrate the need for two-phase sparse matrix multiplication implementations for efficient data structure reuse. Addresses the specific challenges of knowledge graph operations where sparsity patterns and access patterns differ from traditional scientific computing scenarios."
