date: 2025-01-27

summary: "Docling is an easy-to-use, self-contained, MIT-licensed, open-source toolkit for document conversion that can parse several types of popular document formats into a unified, richly structured representation. It is powered by state-of-the-art specialized AI models for layout analysis (DocLayNet) and table structure recognition (TableFormer), and runs efficiently on commodity hardware. Docling is released as a Python package and can be used as a Python API or CLI tool. Its modular architecture and efficient document representation make it easy to implement extensions, new features, models, and customizations. The toolkit has been integrated in popular frameworks like LangChain, LlamaIndex, and spaCy."
